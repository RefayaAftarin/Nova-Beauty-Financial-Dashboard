📊 Financial Performance & Profitability Dashboard | Power BI

📌 Project Overview

This project presents an interactive Financial Performance & Profitability Dashboard built in Microsoft Power BI for a fictional company, Nova Beauty Cosmetics.

The objective of this project is to transform raw financial transaction data into meaningful business insights through data cleaning, modeling, DAX calculations, and interactive visualizations. The dashboard enables business users to monitor sales performance, profitability, expenses, collections, and outstanding payments while supporting data-driven decision-making.

---

🎯 Business Objectives

- Analyze Gross Sales, Net Sales, Gross Profit, and Net Profit
- Monitor Profit Margin and Collection Rate
- Track monthly financial performance
- Compare regional and branch-wise sales performance
- Analyze customer and product category profitability
- Monitor outstanding payments
- Visualize the financial journey from Gross Sales to Net Profit using a Profit Bridge (Waterfall)

---

🧹 Data Cleaning & Transformation (Power Query)

The dataset was cleaned and transformed using Power Query.

### Data Cleaning Tasks Performed

- Converted incorrect date formats using Locale (English - United Kingdom)
- Removed duplicate records
- Trimmed and cleaned all text columns
- Standardized customer names
- Corrected negative quantity values
- Corrected negative outstanding values
- Standardized discount values
- Validated Branch → Region
- Validated Product → Category
- Validated Customer Type
- Removed unnecessary columns
- Changed appropriate data types
- Created reusable validation (Master) tables

💡 Key Business Insights

- Skin Care generated the highest net profit among all product categories.
- Dhaka region contributed the highest net sales.
- March recorded the highest monthly sales and profitability.
- Outstanding receivables remain significant, indicating opportunities to improve collection efficiency.
- Operating expenses account for a relatively small portion of total expenses compared to Cost of Goods Sold (COGS).
